# FlexiZen Frontend (Vite + React)

## Run
1. Install deps:
   ```bash
   cd flexizen-frontend
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```

## Backend
This project assumes Spring backend is running on:
- `http://localhost:8080`

API base proxy is configured in `vite.config.js` for `/api`.
